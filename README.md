# Miami University Triathlon Club Website

A modern, responsive website for the Miami University Triathlon Club, built with React and Tailwind CSS.

## Tech Stack

- [React](https://react.dev/) via [Vite](https://vitejs.dev/)
- [Tailwind CSS v3](https://tailwindcss.com/)

## Setup Instructions

### Prerequisites

- Node.js v18+ and npm
- git

### Installation

```bash
# Clone the repository
git clone <your-repo-url>
cd triathlon-site

# Install dependencies
npm install

# Start the development server
npm run dev
```

Open `http://localhost:5173` in your browser.

### Build for Production

```bash
npm run build
```

Output is placed in the `dist/` folder.
